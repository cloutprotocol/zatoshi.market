# Fonts

## VCR OSD Mono

Download VCR OSD Mono from: https://www.dafont.com/vcr-osd-mono.font

1. Download the font from DaFont
2. Convert to WOFF2 format (use https://everythingfonts.com/ttf-to-woff2 or similar)
3. Place the `VCR_OSD_MONO.woff2` file in this directory

The font is free for personal use. For commercial use, check the license on DaFont.

## Fallback

Until the font is added, the site will use the system monospace font.
